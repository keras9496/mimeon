# 미먼 (MiMeon) — 서비스 소개

> 운영 URL: **https://mimeon.onrender.com/**
> 배포 환경: Render (단일 Web Service, Free 플랜)
> 문서 작성일: 2026-05-12

---

## 1. 한 줄 정의

미먼은 **"내가 평일에 머무는 1~3개 생활공간이 지난 60일 동안 PM2.5·NO₂에 얼마나 위험하게 노출되어 왔는지"** 를 등급(낮음·보통·높음·매우 높음)과 색으로 시각화해 알려주는 웹 서비스입니다.

서비스의 메시지는 **"공기는 뇌에 바로 영향을 준다"** 이며, 미세먼지·이산화질소가 치매·뇌졸중·파킨슨병 위험을 유의하게 높인다는 최신 메타분석(Khreis et al., *Lancet Planetary Health* 2025)을 근거로 합니다.

---

## 2. 누구를 위한 서비스인가

| 사용자 유형 | 미먼이 답을 주는 질문 |
|---|---|
| 일반 시민 | "내가 사는 동네·일하는 사무실의 공기는 뇌 건강 관점에서 안전한가?" |
| 고연령·인지건강 관심층 | "내가 매일 머무는 공간이 치매 위험을 높이고 있지는 않은가?" |
| 가족 돌봄 보호자 | "부모님 댁·요양시설의 공기 노출 수준을 시각적으로 보여줄 수 있을까?" |
| 신경과·예방의학 임상가 | "환자에게 생활환경 노출을 직관적으로 설명할 보조 자료" |

전문가 도구가 아닌 **일반 사용자용 간이 검사** 입니다. 위험도 등급과 색 박스로 결과를 보여주되, 절대 노출 횟수·수치는 일반 사용자가 잘못 해석할 수 있어 의도적으로 노출하지 않습니다.

## 3. 서비스가 제공하는 것 (Value Proposition)

### 3.1 핵심 가치

1. **공공 측정망 데이터를 뇌건강 위험도로 번역**
   에어코리아 측정소가 1시간 단위로 발표하는 PM2.5·NO₂ 값을, 사용자가 실제로 그 시간에 머무는 공간에 한정해 가중 평균한 뒤, 임상연구 기반 위험 등급으로 변환합니다.

2. **사용자 시간·공간 맞춤 분석**
   전국 단위 평균이 아니라, 사용자가 "강남역 사무실 9~18시", "방화동 자택 19~익일8시" 같은 식으로 지정한 **그 위치 × 그 시간대**의 노출만 계산합니다.

3. **위치 간 비교 + 종합 등급**
   1·2·3 위치 각각의 등급뿐 아니라, 시간 가중평균한 **종합 위험도**와 **가장 위험한 위치**를 함께 제시합니다.

4. **지도 위 시각화**
   카카오맵 위에 위치별로 약 666m × 666m 색 박스(50% 투명도, 파랑→빨강 그라데이션)를 그려, 추상적인 숫자가 아닌 **공간 인지**로 위험도를 전달합니다.

5. **HR 기반 20년 누적 치매 위험 + 전국 연평균 비교**
   Khreis 2025 메타분석의 dose-response HR(PM2.5 1.08/5㎍, NO₂ 1.03/10㎍)을 사용자 60일 평균 농도에 적용해 "20년간 같은 노출이 지속될 시 치매 위험 ±X%"를 산출하고, 동일 평균을 환경부 도시대기 측정망 전국 연평균(PM2.5 17㎍/㎥, NO₂ 20ppb)과 비교해 부호 포함 % delta로 함께 제시합니다.

### 3.2 회원가입·로그인이 필요 없음

- 입력한 위치·시간 정보는 요청 처리 중에만 메모리에 존재하고 **DB·로그에 저장하지 않습니다**.
- 결과는 그 자리에서 한 번 보여주고 사라지며, 사용자 추적·재방문 식별을 하지 않습니다.

## 4. 사용자 흐름 (End-to-End)

```
[홈] 평일 주 생활공간 1~3개 등록
      │
      ├─ 박스 클릭 → [검색·시간 모달]
      │     ├─ 카카오맵 키워드 검색 (회사명·건물명·도로명·지번)
      │     └─ 거주 시간대 (시작·종료 시각, 자정 넘김 자동 처리)
      │
      ▼
[CTA] "지난 2달 미먼 위험도 검사하기" 클릭
      │
      ▼
[결과] 좌측: 종합 등급 배지 + 위치별 카드 + 색상 범례
       우측: 카카오맵
             ├─ 위치별 색박스 오버레이
             ├─ 진입 시 위치 1로 줌인 + 노출 경고 카드
             ├─ 좌우 화살표로 위치 1↔2↔3 네비게이션
             └─ 우상단 "전체 보기" 버튼 (전 위치 한눈에)
```

소요 시간: 입력 ~30초, 분석 ~3초(콜드 스타트 시 ~30초), 결과 확인 자유.

## 5. 화면별 제공 기능

### 5.1 홈 화면

- 헤더 라벨 "미먼" + 메시지 **"공기는 뇌에 바로 영향을 준다"**
- 안내 카드 — "내가 생활하는 위치 기준 간이 검사" 설명
- **주 생활 공간 박스 1·2·3**
  - 좌측 색띠: 입력 전 회색, 입력 후 슬롯별 고유 색상
  - 본체: 클릭 시 검색·시간 모달 오픈, 입력 완료 시 주소·시간 표시
- 큰 CTA 버튼 "지난 2달 미먼 위험도 검사하기"

### 5.2 검색·시간 입력 모달

- 키워드 검색 입력란 + 결과 리스트(스크롤)
- 결과 클릭으로 선택, 선택 정보 표시
- 거주 시간 — 시작/종료 시각 드롭다운 (0~23시)
- 자정 넘김(예: 20→08) 안내 텍스트 + 자동 처리
- 다른 슬롯이 이미 점유한 시간은 회색 처리 (시작 시각은 회피, 종료 시각은 경계 허용)
- 저장 / 취소

### 5.3 결과 화면

결과는 보고서 형식(`ReportView`)로 단일 페이지에 표시되며, 상단의 "PDF로 저장" 버튼으로 인쇄·저장이 가능합니다.

**보고서 구성**
- **표지** — 종합 등급 + "지금 노출이 20년 지속될 시 치매 위험 ±X% (HR Y.YY)" verdict
- **01 분석한 생활공간** — 카카오맵 결과 + 위치별 카드(등급 배지 + 위치별 HR/전국비교 sub-row)
- **02 전국 연평균과의 비교** — PM2.5·NO₂ 각각 큰 부호 % delta(↑빨강/↓파랑) + "내 노출 vs 전국 연평균" 농도 좌우 대비
- **03 뇌건강 영향** — 치매 HR / PM2.5 전국대비 / NO₂ 전국대비 3셀 + Khreis 2025 인용
- **04 우울증과의 상호작용** — 동반 위험 안내

**지도 (보고서 01 섹션 내장 카카오맵)**
- 위치마다 색 박스 (Polygon, 약 666m 변, 50% 투명도)
- 중앙 마커
- 진입 시 위치 1로 panTo + 줌 레벨 3 자동 적용
- 좌상단 "미먼 노출 경고" 카드 — 현재 위치의 등급·시간대·측정소
- 가장자리 중앙 **좌우 화살표** — 위치 인덱스 ±1 (경계에서 비활성화)
- 하단 페이지 인디케이터 1/2/3
- 우상단 "전체 보기" 버튼 — 전체 위치 중심 + 줌 레벨 7

## 6. 위험도 산출 방식 (서비스 관점 요약)

### 6.1 임계값

한국 환경부 24시간 "나쁨" 기준을 임계값으로 사용합니다.

| 오염물질 | 임계 | 근거 |
|---|---|---|
| PM2.5 | ≥ 35 ㎍/㎥ | 환경부 대기환경기준 24h 나쁨 |
| NO₂ | ≥ 0.06 ppm | 환경부 대기환경기준 24h 나쁨 |

### 6.2 분석 윈도우

- 종료: **어제 23:59:59** (전일까지)
- 시작: 종료에서 60일 전
- **평일(월~금)만**, 사용자가 지정한 시간대만 포함

### 6.3 지표

각 위치에서, 위 윈도우 안에 들어오는 1시간 단위 측정값에 대해:

```
PM2.5 초과율(%) = PM2.5 임계 초과 시간 / PM2.5 측정 가능 시간 × 100
NO₂   초과율(%) = NO₂   임계 초과 시간 / NO₂   측정 가능 시간 × 100

가중 초과율(%) = (1.0 × PM2.5 초과율 + 0.5 × NO₂ 초과율) / 1.5
```

PM2.5 가중치를 NO₂의 2배로 둔 근거: Khreis 2025 메타분석에서 단위 환산 시 PM2.5의 효과크기가 NO₂ 대비 약 2배 강하기 때문.

### 6.4 등급 체계

| 가중 초과율 | 등급 | 색상 (50% 투명 오버레이) |
|---|---|---|
| < 5% | 낮음 | 파랑 `#2563eb` |
| < 15% | 보통 | 청 `#0ea5e9` |
| < 35% | 높음 | 주황 `#f97316` |
| ≥ 35% | 매우 높음 | 빨강 `#dc2626` |

### 6.5 종합 위험도

```
종합 점수 = Σ (위치별 점수 × 위치별 매칭 시간) / Σ (매칭 시간)
```

시간 가중평균이며, 가장 위험한 위치도 별도 표시.

### 6.6 20년 누적 치매 위험 (HR 기반)

Khreis 2025 메타분석의 dose-response HR(만성 노출 기준)을 60일 평균 농도에 적용합니다.

```
HR_PM2.5 = 1.08 ^ ((PM2.5_avg − 17) / 5)
HR_NO₂   = 1.03 ^ ((NO₂_avg_ugm3 − 41.36) / 10)   # NO₂는 ppm × 1880 으로 단위 환산
HR_total = HR_PM2.5 × HR_NO₂                       # 독립 가정, 보수적 상한
20년 치매 위험 변동 = (HR_total − 1) × 100  (%)
```

기준선은 환경부 도시대기 측정망 **전국 연평균** (PM2.5 17㎍/㎥, NO₂ 20ppb)입니다. 이 값에 정확히 같은 노출이면 HR=1 (0% 변동), 더 깨끗하면 음수, 더 오염되면 양수가 됩니다.

### 6.7 전국 연평균과의 비교

같은 기준선으로 사용자 60일 평균을 직접 비교한 부호 포함 % delta:
```
PM2.5 vs 전국 = (PM2.5_avg − 17) / 17 × 100
NO₂   vs 전국 = (NO₂_avg − 0.020) / 0.020 × 100
```
양수는 빨강(전국 평균 초과), 음수는 파랑(전국 평균 이하)으로 시각화됩니다.

### 6.8 횟수 비공개 원칙

임계 초과 시간 수는 백엔드 응답에 디버깅용으로만 포함되며, **프론트엔드에는 노출하지 않습니다**. 절대 횟수는 사용자가 잘못 해석할 가능성이 커서, 등급과 색으로만 전달합니다.

## 7. 데이터 소스

| 종류 | 출처 | 사용 방식 |
|---|---|---|
| 대기질 측정값 | **에어코리아 OpenAPI** (한국환경공단, 공공데이터포털) | 측정소별 1시간 단위 PM2.5·NO₂, 60일치 |
| 측정소 마스터 | 에어코리아 `getMsrstnList` | 전국 **672개** 측정소 위·경도 (정본 JSON으로 저장소에 커밋) |
| 위치 검색 | **카카오 로컬 API** (Places, Geocoder) | 키워드 → 장소 리스트 + 좌표 |
| 지도 타일 | **카카오 지도 JavaScript SDK** | 색박스 시각화 + 네비게이션 |

**GPS → 측정소 매칭**: 사용자가 선택한 좌표를 ECEF(지구중심좌표계) 3D 벡터로 변환 후, 672개 측정소와 유클리드 거리를 계산해 최단거리 측정소를 선택합니다(~1ms).

## 8. API (서비스가 외부로 노출하는 인터페이스)

| Method | Path | 설명 |
|---|---|---|
| **POST** | `/api/exposure/risk-report` | **v2 메인** — 주 생활공간 1~3개 → 60일 평일 위험도 분석 |
| GET | `/api/health` | 헬스체크 |
| GET | `/api/stations/count` | 로드된 측정소 수 (672) |
| GET | `/api/stations/nearest?lat=&lon=&k=` | 최근접 측정소 k개 |
| GET | `/api/air/station?station_name=&data_term=` | 측정소명으로 측정값 조회 |
| GET | `/api/air/by-gps?lat=&lon=&data_term=` | GPS → 측정소 자동매핑 후 조회 |
| POST | `/api/exposure/analyze` | (legacy v1) GPS 궤적 시간가중 노출 분석 — 프론트 미사용 |
| GET | `/docs` | Swagger UI (자동 생성) |

전체 API 명세는 운영 환경의 `/docs` 페이지에서 직접 확인할 수 있습니다.

## 9. 배포 구성 (Render에서 제공되는 서비스 형태)

- **단일 Web Service** — Render에서 `mimeon` 이라는 이름의 웹 서비스 하나로 배포
- **Uvicorn 단일 프로세스**가 다음 모두를 같은 도메인(`mimeon.onrender.com`)에서 서빙:
  - `GET /` → React SPA (Vite 빌드 결과물)
  - `GET|POST /api/*` → FastAPI 라우터
  - `GET /docs` → Swagger UI
- **빌드 파이프라인**: Vite가 프론트 번들링(162 kB, gzip 53 kB) → pip가 백엔드 의존성 설치 → Uvicorn 시작
- **헬스체크**: `/api/health`
- **플랜**: Free (15분 유휴 시 슬립, 첫 요청 콜드 스타트 ~10-30초)

## 10. 프라이버시·보안 (서비스 사용자에게 약속하는 것)

- **사용자 입력 비저장** — 위치·시간 입력은 요청 처리 동안만 메모리에 존재하며 DB·로그에 남기지 않음
- **에어코리아 API 키 격리** — 백엔드 환경변수에만 존재, 프론트 번들에 미포함 (서버 프록시 호출)
- **카카오 키 도메인 화이트리스트 보호** — 프론트 번들에 인라인되지만 등록된 도메인에서만 작동
- **카카오 검색은 사용자 브라우저 → 카카오 직통** — 백엔드 경유 없음, 사용자 IP/식별 정보가 우리 서버에 남지 않음
- **에어코리아 호출은 백엔드 → 공공데이터포털** — 호출에 사용자 식별 정보 미포함

## 11. 서비스 제약사항 (사용자가 알아야 할 것)

| 제약 | 내용 |
|---|---|
| 조회 기간 | 에어코리아 실시간 API는 최근 **3개월 이내**만. 미먼은 그 중 최근 **60일** 윈도우만 사용 |
| 측정소 결측 | 일부 시점·측정소의 결측 데이터는 분석에서 제외됨 (`matched_hours` 에 미포함) |
| 평일 한정 | 분석은 월~금만, 주말은 별도 입력 미지원 |
| 단일 사용자 가정 | 박스 3개는 한 사람의 1·2·3순위 공간, 다중 사용자 비교 미지원 |
| 위치 수 | 최대 **3개** |
| 시간대 | 0~23시 정수만 입력 가능 |
| API 트래픽 | 공공데이터포털 활용신청 계정의 일일 트래픽 한도 (개발계정 ~500/일) |
| 카카오 무료 한도 | 로컬 검색 30만/일 (사용자 직접 호출이라 백엔드 영향 없음) |
| Cold start | Render 무료 플랜 — 15분 유휴 시 슬립, 첫 요청 10~30초 지연 |

## 13. 한눈에 정리

- **무엇을**: 평일 주 생활공간 1~3개의 PM2.5·NO₂ 위험도 등급화
- **왜**: 공기 노출 → 뇌 건강 위험을 일반인이 직관적으로 이해하도록
- **어떻게**: 에어코리아 60일 시간 단위 데이터 × 사용자 시간·공간 필터 × 임계 초과율 가중 → 4단계 등급 + 카카오맵 색박스 시각화
- **어디서**: https://mimeon.onrender.com/ (Render 단일 웹서비스)
- **무엇으로 빌려쓰는가**: 에어코리아 OpenAPI (측정값), 카카오 지도·로컬 API (검색·지도)
- **무엇을 약속하는가**: 회원가입 없음, 입력 비저장, 절대 노출 횟수는 감추고 등급만 표시
